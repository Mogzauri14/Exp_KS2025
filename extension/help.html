Chrome Extension · How to Use

Global Elections Calendar

A real-time calendar of elections worldwide, fused from 7 international monitoring organisations. Track upcoming votes, check results, and filter by country — all from your browser toolbar.

Getting started
1
Click the extension icon
The globe icon in your Chrome toolbar opens the popup. It loads this month's elections automatically. Data is cached for 30 minutes so it opens instantly on repeat visits.
2
Browse by month
Use the ◀ ▶ arrows at the top to navigate between the previous, current, and next calendar month. The greyed labels beside the arrows show which months are available.
3
Click any election card to learn more
Tapping a card opens the primary source in a new tab. Each card also shows coloured source chips — click a specific chip to open that monitoring organisation's page directly.
Reading a card

Each election is shown as a card like this:

19 APR
Ecuador
Presidential Election (Second Round)
ElectionGuide OSCE/ODIHR Upcoming

The date badge shows the day and month. The source chips list every monitoring organisation that reported this election — the same event from multiple sources is merged into one card automatically.

Tabs: Upcoming · Results · All
Upcoming
Shows only elections that have not yet taken place — status is Upcoming or Unknown.
Results
Shows only elections that have already been held. Use this to quickly find completed votes and check result sources.
All
Shows every election for the selected month regardless of status. Good for a complete overview of the period.
Status meanings
Upcoming Scheduled and not yet held.
Held Election took place; results may be available.
Postponed Originally scheduled but pushed to a later date.
Cancelled Called off entirely.
Disputed Legitimacy or results are contested.
Filtering
Search filter (funnel icon)
Click the funnel icon (≡) next to "Schedule" to reveal a search box. Type any part of a country name, election type, source, or status to filter the list live. Clear the box to show all results again.
Settings — Country filter & pinning

Right-click the extension icon and choose Options to open the Settings page. You can also reach it from chrome://extensions → Details → Extension options.

1
Hide countries you don't need
Uncheck any country and click Save. Unchecked countries are hidden from your feed. Use Select All / Clear All for bulk changes. The search box at the top filters the country list as you type.
2
Pin countries for a focused view
Hover a country row and click the 📌 pin icon. When at least one country is pinned, the popup shows only those countries — useful if you follow a specific region. A "Pinned view · N countries" banner appears at the top of the popup when active. Pins save instantly; no Save button needed.
Tip: Pinning is different from unchecking. Unchecking permanently hides a country. Pinning temporarily focuses only the pinned countries — unpinning them restores the full view without changing your checkbox selections.
Other controls
Refresh ↻
Forces a fresh fetch from the data source, bypassing the 30-minute cache. Use this if you suspect the data is stale.
Help ?
Opens this page any time. Find it in the top-right corner of the popup, next to the refresh button.
Data sources

Elections are cross-referenced from 7 international monitoring bodies and merged into single deduplicated records:

OSCE/ODIHR EEAS (EU) Carter Center ElectionGuide A-WEB IPU Wikipedia
Stale data warning: If the footer turns red and shows "⚠️ WARNING: Stale Data", the underlying data file is more than 48 hours old. The data is still usable but may not reflect very recent announcements.
